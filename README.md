# Advanced SQL Injection Scanner

A comprehensive SQL injection vulnerability scanner with database extraction capabilities, WAF bypass techniques, and detailed reporting.

## Features

- **Multiple SQL Injection Techniques**: Union-based, Error-based, Boolean-based, Time-based
- **WAF Bypass**: Advanced payloads and header manipulation to bypass security filters
- **Database Extraction**: Automatic extraction of databases, tables, and columns
- **External Payload Support**: Load custom payloads from external files
- **Concurrent Scanning**: Multi-threaded parameter testing
- **Comprehensive Logging**: Real-time console output with detailed logging
- **Slow Print Effects**: Professional user interface with typing effects
